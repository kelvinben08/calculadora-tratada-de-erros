# Calculadora Simples com Tratamento de Erros
- Este projeto é uma calculadora em Python capaz de realizar as quatro operações matemáticas básicas: soma, subtração, multiplicação e divisão.

---

### Funcionalidades
- Soma, subtração, multiplicação e divisão
- Uso de funções para cada operação
- Validação da operação digitada pelo usuário
- Tratamento de erros com `try-except`
- Tratamento de `ValueError` e `ZeroDivisionError`

---

### Conceitos praticados
- Funções
- Modularização
- Dicionários
- Estruturas condicionais
- Tratamento de exceções
- Organização de código com `main()`

---

### Como executar
1. Execute o arquivo `main.py`
2. Digite o primeiro número
3. Escolha a operação
4. Digite o segundo número
5. Veja o resultado
